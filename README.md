# LaTeX Fixer

LaTeX Fixer is a desktop application designed to fix errors in LaTeX formulas. It uses LLM to analyze and suggest fixes for formulas, helping users quickly correct errors in mathematical expressions.

## Features

- Real-time formula preview
- Intelligent error detection and fix suggestions
- Support for multiple LaTeX formula formats
- User-friendly interface

## Usage

1. Open the application
2. Input or import your markdown files containing LaTeX formulas
3. View the preview and error list
4. Apply AI-suggested fixes
